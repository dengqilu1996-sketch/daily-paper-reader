## 摘要
足球事件数据为团队运动中球员行为的定量分析提供了丰富的时空来源。这些数据集包含异构特征，将连续的位置坐标与类别变量（如动作类型、动作结果和身体部位）相结合。此类数据已被应用于体育分析中的比赛结果预测、球员评估和战术模式识别。然而，现有方法主要使用独热编码或序数嵌入表示来编码类别特征，忽略了动作描述符的内在语义。Transformer是一种基于自注意力的深度神经网络架构，能够捕捉任意位置输入特征之间的依赖关系。我们提出并实现了一个基于Transformer的模型，以学习类别事件特征间的潜在依赖关系，并生成足球事件的密集表示。通过将类别特征编码为学习的嵌入向量，在预训练过程中捕获了特定于运动的动作语义，使这些表示能够支持动作价值估计和比赛风格识别等下游任务。实证评估表明，在Brier评分测量下，该嵌入表示在下游预测任务上相比特定任务的基线模型产生了更优的概率校准。

## Abstract
Football event data constitute a rich spatiotemporal source for quantitative analysis of player actions in team sports. These datasets contain heterogeneous features, combining continuous location coordinates with categorical variables such as action type, action outcome, and body part. Such data have been applied in sports analytics for match outcome forecasting, player evaluation, and tactical pattern recognition. However, existing approaches predominantly encode categorical features using one-hot or ordinal embedding representations, overlooking the intrinsic semantics of action descriptors. The Transformer is a deep neural network architecture based on self-attention that captures dependencies between input features at arbitrary positions. We propose and implement a Transformer-based model to learn latent dependencies among categorical event features and produce dense representations of football events. By encoding categorical features as learned embedding vectors, sport-specific action semantics are captured during pretraining, enabling the representations to support downstream tasks such as action value estimation and play style recognition. Empirical evaluation shows that the embedding representations yield superior probability calibration over task-specific baselines on the downstream prediction tasks, as measured by Brier score.

## 方法论
### 模型架构：TabTransformer
- **类别特征嵌入**：每个类别特征通过专用嵌入层映射为\(d\)维向量（\(d=32\)）。
- **连续特征处理**：连续变量（如位置坐标、时间戳）保持为标量，与嵌入拼接前不参与Transformer自注意力。
- **自注意力建模**：将类别嵌入序列送入堆叠的\(L\)层Transformer编码器（\(L=6\)，头数8），通过多头自注意力捕捉特征之间的依赖关系（如右脚射门与右路位置的语义组合）。
- **事件表示生成**：将上下文化后的类别嵌入与原始连续特征拼接，得到最终事件表示向量（维度\(K \cdot d + d_{\text{con}} = 911\)）。

### 预训练策略
- **掩码语言建模（MLM）**：以概率0.15随机掩码每个类别特征，模型根据其余未掩码特征预测被掩码值。这迫使模型学习特征间的联合语义。
- **去身份化处理**：预训练前移除球员和球队标识符，防止模型学习到球员声誉等偏差，提高跨联赛可迁移性。
- **排除结果特征**：去除后验信息（如射门终点坐标），避免数据泄露。

### 下游任务适配
- 预训练完成后，事件嵌入可直接用于不同类型任务，无需重新训练编码器，仅需轻量级下游模型（如GRU、MLP、CatBoost）。

## 实验设计
### 数据集
- **来源**：StatsBomb公开数据集。
- **训练集**：2015/16赛季欧洲五大联赛（英超、西甲、德甲、意甲、法甲）共1823场比赛，约6,391,338个事件。
- **测试集**：2020年欧洲杯（51场）、2022年世界杯（64场）、2024年美洲杯（32场）的事件数据，共约57万事件。

### 下游任务与对比方法
1. **Expected Goals（xG）预测**
   - **任务**：给定进攻序列，预测射门是否进球。
   - **模型流程**：预训练TabTransformer生成事件嵌入 → 截断/填充为固定长度序列 → GRU序列建模 → 输出射门得分概率。
   - **对比基线**：MLP (无Transformer嵌入)、StatsBomb官方内置xG模型。
   - **评估指标**：Brier Score、LogLoss为主（衡量概率校准），F1、ROC-AUC为辅。

2. **VAEP（动作价值评估）**
   - **任务**：预测每个持球动作带来的未来10个动作内得分概率变化（得分/丢球两个子任务）。
   - **模型流程**：TabTransformer嵌入 → PCA降维至128维 → CatBoost分类器；同时测试MLP以及直接使用原始特征的XGBoost、CatBoost。
   - **对比基线**：原始VAEP的CatBoost实现（基于SPADL特征）、基于更丰富StatsBomb特征的XGBoost/CatBoost/MLP。
   - **主要指标**：Brier Score、ROC-AUC。

3. **球员风格相似度检索**
   - **任务**：基于单赛季平均事件嵌入，用余弦相似度找出风格最接近的球员。
   - **展示**：为6名精英球员（莱万多夫斯基、梅西、伊涅斯塔、拉莫斯、拉姆、诺伊尔）检索Top-5相似球员，验证位置一致性和语义合理性。

## 主要结论与发现
- **概率校准优势**：基于TabTransformer嵌入的模型在xG（相比MLP）和VAEP（相比多个树模型基线）上取得更低的Brier Score，表明事件嵌入改善了概率校准。
- **通用表示能力**：同一个911维事件嵌入，在不重新训练编码器的情况下，成功应用于序列预测（xG）、表格预测（VAEP）和相似度检索，展现了良好的跨任务迁移性。
- **语义结构挖掘**：球员相似度检索结果显示，平均池化后的事件嵌入能自动发现同位置且风格相近的球员，证实嵌入空间蕴含真实的足球语义。

## 优点
- **统一表示框架**：一次预训练，零样本应用到多种下游任务，大幅降低特征工程和训练开销。
- **特征级自注意力**：针对单事件内的类别特征交互建模，而非事件序列级，能显式学习“动作-身体部位-位置”等组合语义。
- **去偏设计**：去除球员、球队标识，使嵌入聚焦于动作本身，提高跨赛季、跨联赛的泛化性。
- **概率校准导向**：以Brier Score为主要评估指标，契合xG等概率预测任务的实际需求。
