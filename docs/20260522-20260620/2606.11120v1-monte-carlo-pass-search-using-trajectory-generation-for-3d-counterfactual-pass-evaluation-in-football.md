## 摘要
我们将足球中的传球评估重新定义为一个类似于蒙特卡洛树搜索（MCTS）的评估问题，其组成部分在文献中大多已有对应但名称不同：一个价值模型（控球价值）、一个世界模型（多智能体轨迹与球交互），以及一个反事实行动上的策略（通过噪声采样传球变体）。基于首个包含德甲联赛三维球轨迹的公开高保真追踪数据集，我们提出了蒙特卡洛传球搜索（MCPS），该方法推断每次观测到的传球的踢球参数，采样执行变体和选项变体，利用以球为条件的世界模型将每个候选方案向前推演至下一次触球，并使用学习到的价值模型对结果进行评分，从而获得关于所获得价值的分布。该分布使得分布感知的归因成为可能，通过两个互补的执行盈余分数进行：基于均值的分数和基于百分位数的分数，用于分析和排名。为了使世界模型在有限的公开数据下样本高效，我们改编了来自自动驾驶领域的离散标记自回归轨迹生成器（SMART），并表明，与基线相比，该方法在20次采样最优预测准确性上表现出色，同时支持完全假设的推演，以用于下游评估。我们已发布模型检查点和代码。

## Abstract
We recast pass evaluation in football (soccer) as a Monte Carlo Tree Search (MCTS)-like evaluation problem whose components mostly exist in the literature under different names: a value model (possession value), a world model (multi-agent trajectories with ball interactions), and a policy over counterfactual actions (sampling pass variants with noise). Building on the first public high-fidelity tracking dataset with 3D ball trajectories from the Bundesliga, we introduce Monte Carlo Pass Search (MCPS), which infers kick parameters for each observed pass, samples execution variants and option variants, rolls each candidate forward with a ball-conditioned world model until the next ball interaction, and scores outcomes with a learned value model to obtain a distribution over gained value. This distribution enables distribution-aware attribution with two complementary execution-surplus scores used for analysis and ranking: mean-based and percentile-based scores. To make the world model sample-efficient under limited public data, we adapt a discrete-token, autoregressive trajectory generator from autonomous driving (SMART) and show it yields strong best-of-20 forecasting accuracy compared to baselines, while supporting fully hypothetical rollouts for downstream evaluation. We have released model checkpoints and code.

## 1. 研究动机与核心问题

- **现有传球评估的局限**：传统传球评价多基于实际结果（如是否威胁球门），但传球决策受执行噪声（踢球精度）和下游交互（队友争球）双重混淆。一个视觉上相似的传球，可能因微小的执行偏差导致截然不同的结果；而一个技术上完美的传球，也可能因随后的对抗失败而显得低效。这种结果偏差使得**分离选择质量、选项的固有脆弱性以及执行质量**变得困难。
- **点估计的不足**：广受应用的预期威胁（xT）和预期传球（xPass）等指标多为点估计，缺乏对执行敏感性和多模态高质量执行可能性的刻画。物理模型虽然支持假设性分析，但其基于规则的运动假设难以捕捉真实比赛的复杂动态。
- **核心思想**：论文将传球评估重新建模为**蒙特卡洛树搜索（MCTS）类问题**，通过生成大量反事实执行与替代选项，前滚模拟未来交互，并利用学习到的价值函数评分，从而得到**价值分布而非单一值**。这一思路旨在回答：传球者在现有状态下，其实际执行相对于该选项的典型执行以及相对于其他可行选项，是表现更优还是更差？

## 2. 方法论

### 2.1 整体框架：蒙特卡洛传球搜索 (MCPS)
- MCPS 将每次传球评估分解为三个组件：
  - **价值模型**：评估控球状态的价值（控球价值 PV），定义为己方在10秒内进球的概率减去对方得分概率。
  - **世界模型**：在给定传球候选轨迹的条件下，预测多智能体未来运动直至第一次有意义的球接触（触球、解围、出界）。
  - **策略模型（搜索）**：对观测到的传球执行**局部扰动**（同选项不同踢法）和**全局扰动**（不同选项）来生成反事实动作。
- 对每个反事实候选，利用世界模型滚动至首次交互，计算**控球价值增量** \(\Delta PV = PV(s') - PV(s)\)，从而获得该传球的价值分布。

### 2.2 关键组件细节
- **价值模型**：基于Transformer，输入64帧上下文追踪窗口，双头二元交叉熵预测10秒内进球（主客队），用预期进球加权软化标签，训练时随机翻转防止过拟合。
- **世界模型**：
  - 采用离散标记自回归生成器（改编自自动驾驶领域的SMART）。将追踪数据降采样至12.5Hz，一个时间步为5帧。球员运动向量量化为2048码本，球运动向量量化为1024码本，通过k-means构建。解码器仅用空间-时间Transformer，以8步历史预测24步未来，使用掩码交叉熵训练。
  - **触球者预测模块**：基于速度变化和邻近启发式生成接触标签，用生存分析式目标（二元交叉熵危险率+接触者分类）训练。
  - **触球后球速模块**：用对角高斯回归预测触球后球速。
- **传球参数推断与采样**：先通过模拟球物理（重力、阻力等）并最小化轨迹误差，拟合出每次传球的初始三维速度与旋转代理。保留拟合良好的传球（512次）。然后进行**局部扰动**（小幅度方向、速度、旋转）和**全局扰动**（大幅度方向改变）各256次。

### 2.3 评分与归因
- **执行盈余**：定义两个归因指标：
  - **均值差盈余** \(S_{\text{mean}} = \Delta PV_{obs} - \frac{1}{K}\sum_{k}\Delta PV^{(k)}\)（观察值与反事实均值的差）
  - **百分位数盈余** \(S_{\text{pct}} = \frac{1}{K}\sum_{k}\mathbb{I}[\Delta PV_{obs} \ge \Delta PV^{(k)}]\)（观察值在反事实分布中的排名百分位）
- **球员级聚合**：分别按局部搜索和全局搜索计算每位传球手的平均 \(S_{\text{mean}}\) 和平均 \(S_{\text{pct}}\)，前者评估执行稳健性，后者评估选项选择与机会集的差距。
