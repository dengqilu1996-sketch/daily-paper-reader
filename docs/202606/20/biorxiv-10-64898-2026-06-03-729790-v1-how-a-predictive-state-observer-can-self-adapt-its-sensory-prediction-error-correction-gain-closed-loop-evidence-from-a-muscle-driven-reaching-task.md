## 摘要
我们探究了基于前向模型的预测状态观测器在肌肉驱动伸手过程中应如何设定其感觉预测误差校正增益，以及该增益是否能通过智能体可获取的信号——新息历程与每回合伸手结果——而非扫描式神谕标签进行自适应调整。在一个34块肌肉的MyoSuite手臂上，我们评估了残差多层感知机前向模型，任务为低于肩部的逆运动学可达目标，部署于闭环系统，采用使用非负最小二乘肌肉路由和虚拟目标斜坡的稳定末端探头控制器；该控制器是用于评估状态估计效应的稳定探头，而非生物运动规划器。扫描式固定增益闭环神谕揭示了依赖延迟的校正结构：无感觉延迟时，中等校正增益最佳（K = 0.25–0.50），而18步延迟时，观测侧重校正胜出（K = 1.0）。仅前向模型的K = 0消融并非神谕：其系统性差于最佳固定K，差距为1.9–6.1 cm，并显示出由长时域自回归漂移引起的较大非负最小二乘控制器残差；因此我们将K = 0作为诊断报告。结果训练的可靠性自适应观测器在延迟条件下比默认可靠性改善1.9–2.5 cm，同时在无延迟单元中保持中立，而该单元的神谕已为中等。一个特征条件化的β适配器将单元级新息统计映射到每个场的增益参数，在6个单元中的5个几乎匹配了逐单元训练的诊断，但在18步延迟下两者仍比扫描式固定K神谕差1.4–1.8 cm。这些结果分离了依赖延迟的校正结构、K = 0时仅前向模型的失效模式，以及智能体可获取自适应校正的剩余局限性。

## Abstract
We ask how a forward-model-based predictive state observer should set its sensory prediction-error correction gain during muscle-driven reaching, and whether that gain can be adapted from agent-available signals -- innovation history and per-episode reaching outcome -- rather than from swept oracle labels. We evaluate a residual-MLP forward model in a 34-muscle MyoSuite arm on an IK-reachable below-shoulder task, deployed in closed loop with a stabilized endpoint probe controller that uses non-negative least-squares muscle routing and a virtual target ramp; the controller is a stabilized probe for evaluating state-estimation effects, not a biological motor planner. A swept fixed-gain closed-loop oracle reveals a delay-dependent correction structure: with no sensory delay, intermediate correction gains are best (K = 0.25-0.50), whereas with 18-step delay observation-heavy correction wins (K = 1.0). The forward-model-only K = 0 ablation is not the oracle: it is systematically worse than the best fixed K by 1.9-6.1 cm and shows large NNLS controller residuals caused by long-horizon autoregressive drift; we therefore report K = 0 as a diagnostic. Outcome-trained reliability-adaptive observers improve the delayed regime by 1.9-2.5 cm over default reliability while remaining neutral in no-delay cells, where the oracle is already intermediate. A feature-conditioned {beta} adapter that maps cell-level innovation statistics to per-field gain parameters nearly matches a per-cell trained diagnostic in 5/6 cells, but both remain 1.4-1.8 cm worse than the swept fixed-K oracle at 18-step delay. These results separate the delay-dependent correction structure, the forward-model-only failure mode of K = 0, and the remaining limits of agent-available adaptive correction.

*   **研究背景与动机**：生物运动控制中，中枢神经系统需利用前向模型和感觉反馈估计状态，但其校正权重并非固定。传统自适应滤波多基于噪声协方差传播，本工作则跳出此框架，探讨在闭环、任务驱动的场景下，能否从“智能体可用”的简单信号中学习到有效的校正策略。这具有明确的仿生动机：生物体无法获取最优增益的真值，只能依靠自身的感觉预测误差和动作结果进行调节。
*   **整体含义**：该研究本质上是对预测状态观测器校正增益的**闭环特性和自适应能力**进行系统性的计算基准测试与诊断，旨在分离延迟依赖的增益结构、仅靠前向模型的失效模式，以及当前自适应策略的剩余局限，为理解生物运动状态估计的计算原理提供闭环证据。

### 2. 论文提出的方法论：核心思想、关键技术细节
论文的核心方法论由四个层面构成：前向模型、预测状态观测器、可靠性自适应增益规则、跨回合结果自适应。

*   **前向模型**：
    *   采用**残差多层感知机（Residual MLP）** 预测状态差分 \(\Delta x_t = x_{t+1} - x_t\)，而非直接预测下一状态，利于容量聚焦于动态变化。
    *   输入为83维状态和34维肌肉兴奋，架构为两层256单元ReLU+LayerNorm网络。
    *   **多步展开监督训练**：为应对长时域闭环稳定性，采用\(H=8\)步的自回归展开损失进行训练，使其在长距离预测下更鲁棒。
*   **固定滞后预测状态观测器**：
    *   采用**新息更新**（innovation-update）形式：\(\hat{x}_{t-d} \leftarrow \hat{x}_{t-d} + K \odot (y_t - \hat{x}_{t-d})\)，其中 \(y_t\) 是带延迟 \(d\) 和噪声的观测，\(\hat{x}_{t-d}\) 是延迟步的预测估计。
    *   通过环形缓冲区存储过去估计与动作，将修正后的过去估计重新前向展开至当前时刻，得到 \(\hat{x}_t\)。
    *   校正增益 \(K\) 在整个论文中先被当作标量（同播至所有83维状态），后在自适应部分被扩展为**场方向量**（将83维状态分为5个感觉场：关节位置、速度、肌肉激活、指尖位置、伸手误差，每场独立设 \(K_f\)）。
*   **可靠性自适应增益规则（回合内层）**：
    *   **核心思想**：利用感觉通道自身的**新息历史**在线估计当前感觉的可靠性 \(r_f(t)\)，并据此动态调节校正增益 \(K_f(t)\)。
    *   **具体机制**：对每个感觉场 \(f\)，用指数加权移动平均（EMA）估计逐元素平方新息的均值 \(v_f(t)\)，其倒数作为可靠性估计 \(r_f(t) = 1/(\epsilon + v_f(t))\)。
    *   **增益映射**：通过两参数逻辑斯谛函数将可靠性映射到增益：\(K_f(t) = \sigma(\beta_{0,f} + \beta_{1,f} \log r_f(t))\)。其中，\(\beta\) 参数决定了增益对可靠性变化的敏感度和工作点。默认设为 \(\beta_{0,f}=0, \beta_{1,f}=0.5\)，使得增益起始于0.5左右，并在新息减小时（传感器可靠）增大，新息增大时（传感器噪声大）减小。
*   **跨回合结果自适应（回合间层）**：
    *   使用**同时扰动随机逼近（SPSA）** 算法，以每回合最小指尖误差 \(\text{minTip}\) 作为结果信号，优化全局的10维元参数 \(\beta\)（5个场×2个参数）。
    *   SPSA仅需两点评测扰动来估计梯度，无需可微模型，且代价与 \(\beta\) 的维度（10）而非状态维度（83）挂钩。
*   **特征条件化 \(\beta\) 适配器**：
    *   进一步探索：能否从当前噪声/延迟“单元”的在线创新统计特征（如新息的均值、方差）出发，直接输出 \(\beta\) 的修正量 \((\Delta \beta_{0,f}, \Delta \beta_{1,f})\)，实现**跨情景泛化**。
    *   使用一个30参数的线性适配器（每场2个输出×3个权重），将经过z-score归一化的对数新息统计量映射到 \(\beta\) 的加性修正上。

### 3. 实验设计
*   **环境与任务**：MyoSuite `myoArmReachFixed-v0` 任务，在 MuJoCo 中模拟34块肌肉的肩-肘-腕-手系统。任务目标是驱动食指尖到达固定目标，单回合长12秒（600控制步）。
*   **目标集**：采样自任务空间，仅保留**低于肩部**（\(z<1.393m\)）、逆运动学可达的目标（n=200个），以模拟桌平面伸手并避免力学奇点。
*   **控制器**：采用**稳定的末端探头控制器**，包含任务空间PD控制、通过非负最小二乘（NNLS）将关节力矩映射到非负肌肉激活、以及虚拟目标斜坡（防止初始冲击）。该控制器被明确界定为评估状态估计效应的“探头”，而非生物运动规划模型。
*   **评估基准**：
    *   **扫描固定增益神谕**：在3种噪声水平（无噪声、高、极高）×2种延迟（0步、18步）的6格网格中，对 \(K \in \{0, 0.25, 0.5, 0.75, 1\}\) 每个值进行200回合闭环实验，得到各单元下的最佳固定 \(K\) 和最小指尖误差。
    *   **诊断基准**：\(K=0\)（仅前向模型）作为失败模式诊断；每种噪声/延迟单元内的**最佳固定 \(K\)** 作为性能上界（神谕）。
*   **对比方法**：
    *   **默认可靠性规则**：\(\beta_{0,f}=0, \beta_{1,f}=0.5\)，无训练。
    *   **全局SPSA \(\beta\)**：在所有6个细胞上通过SPSA训练单一组 \(\beta\)，部署到各细胞。
    *   **特征条件化 \(\beta\)**：在全局SPSA \(\beta\) 基础上，通过30参数适配器根据细胞特征在线调整 \(\beta\)。
    *   **逐细胞SPSA诊断**：在每个细胞内部单独训练一组 \(\beta\)，代表本参数化能力上界（非智能体现实策略）。
    *   所有方法均与 **扫描固定K神谕** 和 **K=0** 诊断进行对比。

### 6. 论文的主要结论与发现
*   **发现1：最优增益依赖延迟**。固定K扫描神谕揭示，在无延迟时最优K为中等（0.25-0.50），且随噪声增大趋向前向预测；但在**18步延迟下，最优K坍缩为1.0**（完全信任观测），即观测侧重校正胜出，因为此时前向预测本身也已陈旧。
*   **发现2：K=0是失效诊断，非神谕**。仅前向模型（K=0）在**所有细胞**中系统性差于最佳固定K（差距1.9-6.1 cm），并引发巨大NNLS控制器残差（92-96 vs. <50），原因是长时域自回归漂移到非物理状态。因此，**K=0应被视为暴露前向模型闭环缺陷的诊断工具，而非可用的最优策略**。
*   **发现3：结果驱动的自适应在延迟下有效，无延迟无效**。基于SPSA训练的单一全局\(\beta\)，在**所有延迟-18细胞**中均比默认规则显著改善（1.85-2.46 cm），配对bootstrap置信区间全部排除零；而在**无延迟细胞**中，改善统计不显著。这完美呼应了发现1：当神谕是K=1时，自适应能朝向它；当神谕为中等K时，默认规则已接近，难以提升。
*   **发现4：当前参数化下自适应仍存不可逾越的差距**。即使最强的自适应方案（特征条件化\(\beta\)和逐细胞训练\(\beta\)），在延迟-18细胞中仍**比扫描固定K神谕差1.4-1.8 cm**（CIs排除零）。这表明可靠性-增益逻辑斯谛函数参数化、细胞级新息特征以及固定滞后观测器动力学共同构成了当前自适应能力的上限。

### 8. 不足与局限
*   **前向模型与目标漂移缺陷**：论文坦承，由于前向模型预测含目标场的完整状态，在零动作长时域展开下会出现目标漂移，这可能影响闭环K=0时的稳定性评估。已列为未来改进（如目标场钳位或移除）。
*   **控制器非生物性**：所用“稳定探头控制器”是精心设计的评估工具，但并非生物运动控制器模型，这限制了将发现直接推广到生物启发的运动规划控制器的结论效度。
*   **任务场景泛化性有限**：仅测试了低于肩部的固定目标伸手任务，未在更广泛的目标集（如全肩部空间、随机运动轨迹）上验证。
*   **自适应参数化空间受限**：关键局限即发现4所述：当前简单的场向逻辑斯谛增益规则和低维特征映射可能无法足够表达最优增益策略，切换更丰富的参数化方案（如神经网络映射）可能会缩小剩余差距，但论文尚未探索。
*   **延迟值为特例**：仅测试了0步和18步两种离散延迟，未研究延迟长度连续变化时自适应策略的迁移性。
*   **单一随机种子**：虽然闭环评估有200回合保障，但SPSA训练和适配器训练仅基于单一随机种子，可能引入一定随机性偏差，尽管论文指出可通过种子复现。
