## 摘要
无监督视频对象跟踪旨在将动态场景分解为持久的、以对象为中心的实体，无需手动标注。近期许多方法依赖于基于槽位的表示，其中一组固定的潜在变量（“槽位”）在帧间表示各个对象。为保持对象身份，这些模型在槽嵌入上施加时序一致性。然而，当外观和姿态纠缠在一起时，这种一致性目标与对象运动和视角变化相冲突。因此，槽位倾向于锁定在静态区域（如背景）以满足一致性目标，而前景对象则被分割到多个槽位中或频繁交换身份。为解决这些限制，我们提出STAITUS，一个统一框架，显式地将每个槽位解耦为外观和几何姿态（位置/尺度）。利用这种解耦，STAITUS在帧内施加空间分离，并仅在外观空间中进行时序对齐，从而在运动、遮挡和对象进入/离开情况下获得更清晰的分割掩码和更持久的身份。此外，为缓解过分割，我们引入了一种自适应门控机制，动态调整活跃槽位的数量以适应场景复杂性。在合成和真实基准上的大量实验表明，STAITUS在分割质量和跟踪稳定性方面显著优于最先进的基线方法。

## Abstract
Unsupervised video object tracking aims to decompose dynamic scenes into persistent, object-centric entities without manual annotations. Many recent approaches rely on slot-based representations, where a fixed set of latent variables ("slots") represent individual objects across frames. To preserve object identity, these models enforce temporal consistency on slot embeddings. However, when appearance and pose are entangled, this consistency objective conflicts with object motion and viewpoint changes. As a result, slots tend to lock onto static regions (e.g., background) to satisfy the consistency objective, while foreground objects become fragmented across multiple slots or frequently swap identities. To address these limitations, we propose STAITUS, a unified framework that explicitly disentangles each slot into appearance and geometric pose (position/scale). Leveraging this disentanglement, STAITUS enforces within-frame spatial separation and applies temporal alignment only in appearance space, yielding sharper masks and more persistent identities under motion, occlusion, and object entry/exit. Furthermore, to mitigate over-segmentation, we introduce an adaptive gating mechanism that dynamically adjusts the number of active slots to match scene complexity. Extensive experiments on synthetic and real-world benchmarks demonstrate that STAITUS substantially outperforms state-of-the-art baselines in segmentation quality and tracking stability.

*   **核心问题**：现有基于槽位（Slot）的模型为保持对象身份，通常强制槽嵌入在时序上保持一致。然而，槽嵌入通常同时编码对象的外观和空间姿态（位置、尺度），导致**外观-姿态纠缠**。
*   **冲突根源**：强制时序一致性时，模型倾向于让槽位锁定在静态背景区域（易于满足一致性），而运动的前景对象则因姿态变化导致嵌入变化，从而出现**对象碎片化**（一个物体被多个槽位表示）和**身份交换**（槽位与物体的对应关系跳变）。
*   **整体含义**：论文指出，在姿态纠缠的表征上施加时序一致性是现有方法不稳定跟踪的**根本失效模式**。提出通过显式解耦外观与姿态，并仅在外观空间施加对齐，可从根本上解决这一冲突。

### 2. 方法论：核心思想、关键技术细节与算法流程
*   **核心思想**：提出**STAITUS** (Sparse and Temporally Aligned InvarianT Unsupervised Slots) 框架，将每个对象的表示显式拆分为**外观嵌入**（$v_t^k$）和**几何姿态**（2D位置 $p_t^k$、尺度 $s_t^k$）。让姿态可自由演化，仅在外观空间施加时序一致性。
*   **关键模块与流程**：
    1.  **密集特征提取**：使用冻结的预训练DINO骨干网络提取帧的密集特征，经一个小型MLP投影到任务特征空间。
    2.  **循环自适应解耦槽注意力**：
        *   **解耦分组**：基于Invariant Slot Attention (ISA)，利用每个槽自身的姿态构建参考系，从姿态归一化的特征中提取外观嵌入 $v_t^k$，使其对运动不变。
        *   **时序初始化**：用前一帧的槽状态预测当前帧的初始槽。外观采用高斯分布预测以适应缓慢变化，姿态则直接传递（平稳运动假设）。
        *   **自适应槽激活**：引入门控MLP $G_{gate}$，根据外观嵌入 $v_t^k$ 动态预测二值激活变量 $z_t^k$，决定哪些槽参与重建，从而自动匹配场景中的物体数量，减少过分割。训练时用Gumbel-Softmax松弛。
    3.  **自适应空间广播解码**：每个活跃槽独立解码生成RGB重建和Alpha掩码。解码器使用槽自身的姿态参数构建相对坐标网格，得到精细掩码。最终帧重建由所有活跃槽的输出合成。
*   **损失函数设计**：
    *   **重建损失** ($L_{recon}$)：像素级MSE，作为无监督主信号。
    *   **时序对齐损失** ($L_{time}$)：仅对外观嵌入 $v$，在连续帧间最大化活跃槽的余弦相似度。
    *   **空间分离损失** ($L_{sep}$)：在同一帧内，惩罚活跃槽外观嵌入之间的正余弦相似度，防止多个槽塌缩到同一物体或区域。
    *   **稀疏正则** ($L_{spars}$)：鼓励模型只激活必要数量的槽。
    *   **总损失**：$L = L_{recon} + \lambda_{time}L_{time} + \lambda_{sep}L_{sep} + \lambda_{spars}L_{spars}$。
