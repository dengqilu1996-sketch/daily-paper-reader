## 摘要
分析细粒度技能活动（如体育、手术）不仅需要识别视觉模式，还需执行逐步的视觉推理以得出最终判断。尽管动作质量评估的最新进展在评估性能方面取得了显著进步，但现有模型仍是黑箱，缺乏揭示其判断背后推理过程的能力。为解决这一局限，我们提出了潜在视觉扩散推理（Latent Visual Diffusion Reasoning，LVDR），一个新颖的框架，它集成关键点引导的蒙特卡洛树搜索（Monte Carlo Tree Search，MCTS），对潜在视觉推理过程进行建模和可视化。LVDR不仅能产生更准确的技能评估，还能揭示对最终评价有贡献的关键视觉推理序列。在涵盖多样体育和手术领域的四个数据集上的大量实验表明，LVDR实现了有竞争力的量化性能，同时提供了通向最终预测的可解释视觉推理轨迹。源代码和模型可通过以下链接获取：https://github.com/XiruiTeng/LVDR_Official.git。

## Abstract
Analyzing fine-grained skill activities (e.g., sports, surgery) requires not only recognizing visual patterns but also performing step-by-step visual reasoning that leads to the final judgment. While recent advances in action quality assessment have achieved remarkable progress in evaluating performance, existing models remain black boxes, where they lack the ability to explicitly reveal the reasoning processes underlying their judgments. To address this limitation, we propose Latent Visual Diffusion Reasoning (LVDR), a novel framework that integrates keypoint-guided Monte Carlo Tree Search (MCTS) to model and visualize the latent visual reasoning process. LVDR not only produces more accurate skill assessments but also uncovers the critical visual reasoning sequences that contribute to the final evaluation. Extensive experiments across four datasets spanning diverse sports and surgical domains demonstrate that LVDR achieves competitive quantitative performance while providing interpretable visual reasoning trajectories leading to the final predictions. Source codes and models can be found through the following link: https://github.com/XiruiTeng/LVDR_Official.git.

### 2. 论文提出的方法论

LVDR框架主要由两个核心组件构成：

*   **核心思想**：将视频的逐步视觉推理过程建模为一个从噪声（无信息先验）到目标语义分布（视频整体语义）的渐进式去噪扩散过程。同时，通过MCTS在人体关键点构成的组合搜索空间中，寻找并解释对最终评分最具影响力的推理路径。

*   **潜在视觉扩散推理（Latent Visual Diffusion Reasoning）**
    *   **技术细节**：
        *   该模块基于去噪扩散隐式模型（DDIM）构建。它将完整视频的目标语义分布（从专家评论中提取的文本嵌入 `zc`）作为去噪目标。
        *   对于每个视频帧 `Ii`，提取其3D关键点 `Ki`，通过一个“推理状态学习器” `θR` 生成初始的视觉推理嵌入 `Pi`。
        *   利用一个扩散Transformer模型 `θD`，将前一帧的推理嵌入 `P_{i-1}` 作为输入，以当前帧的初始嵌入 `Pi` 为条件，逐步去噪，生成更为精炼的当前帧推理嵌入 `P_i`。
    *   **公式与算法流程**：
        *   **训练目标**：模型训练包含两个损失函数：
            *   **扩散损失** `LDM`：标准的DDIM去噪损失，训练模型学习预测所添加的噪声。
                `LDM = E_{ε~N(0,1), t, P_t} [|| ε - ε_θ(P_t, t, P_t) ||^2]`
            *   **评分回归损失** `Lscore`：利用最终帧的推理嵌入 `P_T`，通过全连接层预测得分 `s`，并计算与真实得分 `s` 的均方误差。
                `Lscore = || s - s ||^2`
            *   **总损失** `L = LDM + λLscore`，其中 `λ` 为权重超参数（论文中设为1）。

*   **关键点引导的蒙特卡洛树搜索（MCTS）**
    *   **核心思想**：模拟人类裁判在评估时系统性地关注不同身体部位的过程。将这种视觉处理策略建模为在关键点构成的组合搜索空间中进行MCTS。
    *   **技术细节**：
        *   **状态与动作定义**：MCTS的每个节点代表一个“状态”，即当前关注的关节组合。
            *   **体育场景**：动作空间为四大关节：肘部、肩部、髋部、膝部。
            *   **手术场景**：动作空间为器械的三大部件：尖端、主体、尾部。
        *   **节点表示**：每个节点由其对应关节的物理信息表示：`(角度a，速度v，3D位置p)`。
        *   **推理与可视化**：在推理时，对每一帧视频的潜在推理嵌入执行MCTS。通过选择、扩展、模拟（利用扩散模型的输出来计算余弦相似度作为奖励）、反向传播四个阶段迭代搜索。经过多次迭代后，选择拥有最高期望奖励的路径 `v*`，这条路径所对应的关键点序列即为该帧最关键的视觉依据，其被选中的频次被可视化为不同的颜色，展示了模型的注意力模式。
